NuTitleCase
===========

NuTitleCase is an implementation of [TitleCase][sp] in [Nu][nu].

[sp]: http://daringfireball.net/2008/05/title_case
[nu]: http://programming.nu

NuTitleCase may be built as a framework using `nuke`.
